Max

The software Max is quite extensive and allows you to develop your own ideas, that aren't constricted by the design of the program. It also allows for extendability being able to communicate with other audio interface technologies, it also allows Javascript.

I look forward to using Max more, as a student studying Programming the same concept are applied in Max. Just that Max does a lot of the hard work for you. A cool feature i like about Max is that it is easy to share code snapshots of linked objects. You can easily do this by converting the selection to Max based text format that can be copy pasted to be shared. This is very useful when displaying code in a forum, the other user just needs to paste from clipboard in Max to display a patch.

Stretta is one of my favorite users of Max language, he combines a very artistic background intertwined with technological knowledge.
